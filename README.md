# Testing performance of Require.js vs. Vanilla JS

## Why?

There has been misconception about Require.js' performance. Please read more in
my artice.

## Install and run

You might want to visit `index.html`. Before you do that please run

    $ npm i

and make sure you've Uglify-JS installed globally

    $ npm i -g uglify-js

After that just run a compile of the sources:

    $ npm run compile
